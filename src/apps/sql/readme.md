# SQL

SQL（Structured Query Language，结构化查询语言）是一种用于管理和操作关系型数据库的标准化编程语言。

在本教程中，您将学到如何使用 SQL 访问和处理数据系统中的数据，这类数据库包括：MySQL、SQL Server、Access、Oracle、Sybase、DB2 等等。

# 工具说明

- 左边输入源数据，右侧可选择 2，4，Tab 的缩进方式格式化 JSON。也可以选择压缩 JSON
